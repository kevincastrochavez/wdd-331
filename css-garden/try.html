Please view page on a desktop browser to see parallax effect

This is a page I created to experiment with the parallax effect
and test applying the effect in a practical way like in a website header.
I created 4 different images in photoshop and then layered and spaced them accordingly to achieve the desired effect.
I then used the "transform: translateZ" CSS attribute on each layer to adjust the amount each layer moved when scrolling down the page.
I am currently working on a way to scale it down for mobile use
